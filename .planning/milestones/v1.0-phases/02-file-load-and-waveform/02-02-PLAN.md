  <action>
Create `src/components/FileLoader.tsx`:
- Import `useTrimStore` from store, `checkFileFormat` and `MAX_FILE_SIZE` from formatValidation, `decodeForWaveform` from audioDecoder
- Define `loadAudioFile` async function (can be inside component or module-level using `useTrimStore.getState()`):
  1. Call `checkFileFormat(file)` — if not valid, call `setStatus('error', check.reason)` and return
  2. Check `file.size > MAX_FILE_SIZE` — if true, call `setStatus('error', 'File is too large (max 50 MB). Select a shorter recording.')` and return
  3. Call `setFile(file)` (sets status to 'decoding')
  4. Try: `const arrayBuffer = await file.arrayBuffer()`, then `const audioBuffer = await decodeForWaveform(arrayBuffer)`, then `setAudioBuffer(audioBuffer)`
  5. Catch: `setStatus('error', 'Could not decode audio: ' + error message)`
- Render a styled button/label wrapping a hidden `<input type="file" accept=".webm,audio/webm,video/webm">`
- On change: get `e.target.files?.[0]`, if exists call `loadAudioFile(file)`, then reset `e.target.value = ''` so same file can be re-selected
- Show "Decoding..." or similar text when status is 'decoding'

Create `src/components/WaveformView.tsx`:
- Import `WaveSurfer` from `wavesurfer.js`, `useRef`, `useEffect` from React, `useTrimStore` from store
- Create `containerRef = useRef<HTMLDivElement>(null)` and `wsRef = useRef<WaveSurfer | null>(null)`
- Read `audioBuffer` from `useTrimStore(state => state.audioBuffer)`
- `useEffect` with `[audioBuffer]` dependency:
  - If no containerRef.current or no audioBuffer, return
  - Call `wsRef.current?.destroy()` BEFORE creating new instance (prevents memory leak / visual corruption)
  - Create: `wsRef.current = WaveSurfer.create({ container: containerRef.current, waveColor: '#4a9eff', progressColor: '#1a6fc4', height: 128, interact: false })`
  - Call `wsRef.current.loadDecodedBuffer(audioBuffer)`
  - Cleanup function: `wsRef.current?.destroy(); wsRef.current = null`
- Render: `<div ref={containerRef} />` with appropriate styling (min-height, border, etc.)
- Only render container div when audioBuffer exists (or show placeholder text when idle)
  </action>
